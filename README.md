# Isa Hussain - Portfolio v2

This project is a Next.js portfolio website built with the following stack:

- **Framework:** Next.js 16 (App Router, TypeScript, static export)
- **Styling:** Tailwind CSS v4 + CSS custom properties
- **Scroll:** Lenis
- **Animations:** GSAP + ScrollTrigger
- **Transitions:** Motion (Framer Motion)
- **Fonts:** next/font (Google Fonts — Cormorant Garamond + Inter)
- **Video:** HTML5 `<video>`
- **Contact:** React form → PHPMailer + SMTP
- **Deployment:** `next build` → upload `/out` + `/api` to cPanel

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
